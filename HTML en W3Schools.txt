=== INTRODUCCION A HTML ===

¿QUE ES HTML?

HTML significa HyperText Markup Language, es el lenguaje estandar
para la creacion de paginas web, describe lan estructura
de una pagina web

Consta de una serie de elementos, que le dicen al navegador
como mostrar el contenido, etiquetando fragmentos del contenido
comon "este es un encabezado", "esto es un parrafo", etc.

UN DOCUMENTO HTML SENCILLO

Ejemplo

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Ejemplo explicado

El "<!DOCTYPE html>" es la declaraciop que define que este
documento es un documento HTNL5

El <html> es el elemento raiz de una pagina HTML 

El <head> es el elemento que contiene metainformacion
sobre la pagina HTML

El <title> es un elemento que define el cuerpo del documento
y en un contenedor para todo el contenido visible, como por ejemplo
encabezados, parrados, imagenes, hipervinculos, tablas, listas, etc.

El <h1> es el elemento que define un encabezado grande.

El <p> es el elemento que define un parrafo.

Un elemento HTML se define mediante una etiqueta de inicio, algun contenido
y una etiqueta final.

<tagname> El contenido va aquí... </nombre de etiqueta>

El elemento HTML es todo desde la etiqueta inicial hasta la etiqueta final

<p>Mi primer parrafo</p>

DOCUMENTOS HTML

Todos los documentos HTML deben comenzar con una declaracion de tipo de documento:
<!DOCTYPE html>.

El documento HTML en sí comienza con <html> y termina con </html>.

La parte visible del documento HTML está entre <body> y </body>.

EJEMPLO

<!DOCTYPE html>
<html>
<body>

LA DECLARACION <!DOCTYPE>

La declaracion <!DOCTYPE> representa el dipo de documento
y ayuda a los navegadores a mostrar las páginas web correctamente.

Sólo debe aparecer una vez, en la parte superior 
de la página (antes de cualquier etiqueta HTML).

La declaración <!DOCTYPE> no distingue entre mayúsculas y minúsculas.

La declaración <!DOCTYPE> para HTML5 es:

<!DOCTYPE html>

=== ELEMENTOS HTML ===

Un elemento html se define mediante una etiqueta de inicio, algun contenido
y una etiqueta final.

El elemento html es todo desde la etiqueta inicial hasta la etiqueta final

Ejemplos de algunos elementos HTML:

<h1>Mi Primer título</h1>
<p>Mi primer párrafo.</p>

ELEMENTOS ANIDADOS HTML

Los elementos HTML se pueden anidar 
(esto significa que los elementos pueden contener otros elementos).

Todos los documentos HTML constan de elementos HTML anidados.

El siguiente ejemplo contiene cuatro elementos HTML (<html>, <body>, <h1> y <p>):

EJEMPLO

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

MI EJEMPLO

<!DOCTYPE html>
<html>
<body>

<h1>ELEMENTOS ANIDADOS</h1>
<p>PRIMER PARRAFO</p>

</body>
</html>

Ejemplo explicado

El <html> El elemento es el elemento raíz y define todo el documento HTML.
Tiene una etiqueta de inicio <html> y una etiqueta final </html>.
Luego, dentro del <html> elemento hay a <body> elemento:

<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>

El <body> el elemento define el cuerpo del documento.
Tiene una etiqueta de inicio <body> y una etiqueta final </body>.
Luego, dentro del <body> elemento allí son otros dos elementos: <h1> y <p>:

<h1>My First Heading</h1>
<p>My first paragraph.</p>

El <h1> El elemento define un encabezado.
Tiene una etiqueta de inicio <h1> y una etiqueta final </h1>:

<h1>My First Heading</h1>

El <p> El elemento define un párrafo.
Tiene una etiqueta de inicio <p> y una etiqueta final </p>:

<p>My first paragraph.</p>

NUNCA OMITAS LA ETIQUETA FINAL 

Algunos elementos HTML se mostrarán correctamente, 
incluso si olvidas la etiqueta final:

<html>
<body>

<p>This is a paragraph
<p>This is a paragraph

</body>
</html>

Sin embargo, ¡nunca confíes en esto! 
¡Pueden producirse resultados y errores inesperados si olvidas la etiqueta final!

ATRIBUTO SRC

La etiquete img, se utiliza para incrustar una imagen en una pagina HTML.
El atributo SRC especifica la ruta a la imagen que se mostrara.

EJEMPLO

<!DOCTYPE html>
<html>
<body>

<h2>The src Attribute</h2>

Mi EJEMPLO

<!DOCTYPE html>
<html>

</body>
</html>

Existen dos formas de especificar la irl en el atributo src:

1. URL Absoluta
    Enlaces a una imagen externa alojada en otro sitio web

2. URL Relativa
    Enlaces a una imagen alojada en El sitio web. Aquí, la URL no incluye 
    el nombre de dominio. Si comienza la URL Sin barra, será relativa 
    a la página actual. Ejemplo: src="img_girl.jpg". 
    Si la URL comienza con una barra, será relativa al dominio. 
    Ejemplo: src="/images/img_girl.jpg".

ATRIBUTOS DE ANCHO Y ALTO

La etiqueta img tambien debe contener los atributos width y height,
que especifivan el ancho y altura de la imagen (en pixeles).

<img src="img_girl.jpg" width="500" height="600">

ATRIBUTO ALT

El atributo alt es requerido para la etiqueta img, especifica un texto
alternativo para una imagen, si por alguna razon, la imagen no se puede mostrar.
Esto puede deberse a una conexion lenta o un error en el atributo src, o
si el usuario utiliza una pantalla de inicio.

<img src="img_girl.jpg" alt="Girl with a jacket">

Esto es lo que sucede si intentamos mostrar una imagen que no Existe

<img src="img_typo.jpg" alt="Girl with a jacket">

ATRIBUTO STYLE

El atributo STYLE se utiliza para agregar 
estilos a un elemento, como color, fuente, tamaño y más.

<p style="color:red;">This is a red paragraph.</p>

ATRIBUTO LANG

Siempre se debe incluir el atributo LANG dentro de la etiqueta HTML
para declarar el idioma de la página web. Esto está destinado a 
ayudar a los motores de búsqueda y navegadores.

El siguiente ejemplo especifica el inglés como idioma:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

También se pueden agregar códigos de país
al código de idioma en el atributo LANG. 
Entonces, los dos primeros caracteres definen 
el idioma de la página HTML, y los dos últimos 
personajes definen el país.

El siguiente ejemplo especifica inglés como idioma 
y Estados Unidos como el país:

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

ATRIBUTO TITLE

El atributo TITLE define algo extra información sobre un elemento.
El valor del atributo de título se mostrará como información sobre 
herramientas cuando pasas el ratón sobre el elemento:

<p title="I'm a tooltip">This is a paragraph.</p>

SUGERENCIAS

UTILIZAR SIEMPRE ATRIBUTOS EN MINUSCULAS

El estandar de HTML no requiere nombres de atributos en minuscukas.
Todos los atributos se pueden escribir en mayusculas o minusculas como
tytle o TYTLE

Sin embargo, W3C recomienda atributos en minusculas en HTML.

SIEMPRE CITA VALORES DE ATRIBUTOS

El estándar HTML no requiere comillas alrededor de los valores de los atributos.

Sin embargo, W3C recomienda citas en HTML, y demandas 
citas para tipos de documentos más estrictos como XHTML.

BIEN

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>

MAL

<a href=https://www.w3schools.com/html/>Visit our HTML tutorial</a>

A veces hay que utilizar comillas. 
Este ejemplo no se mostrará el atributo de título 
correctamente, porque contiene un espacio:

<p title=Description of W3Schools></p>

¿CITAS SIMPLES O DOBLES?

Las comillas dobles alrededor de los valores de los atributos son los mas
comunes en HTML, pero las simples tambien se pueden utilizar en citas.

En algunas situaciones, cuando el balor del atributo en si contiene comillas dobles,
es necesario utilizar comillas simples.

<p title='John "ShotGun" Nelson'></p>

O viceversa:

<p title="John 'ShotGun' Nelson">

=== PARRAFOS HTML ===

Un párrafo siempre comienza en 
una nueva línea y suele ser un bloque de texto.

PARRAFOS EN HTML

El elemento html p define un parrafo

Un párrafo siempre comienza en una nueva línea y los navegadores 
agregan automáticamente algún espacio en blanco (un margen) 
antes y después de un párrafo.

Ejemplo

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

No puedes estar seguro de cómo se mostrará HTML.

Las pantallas grandes o pequeñas y las ventanas
redimensionadas crearán resultados diferentes.

Con HTML, no puedes cambiar la visualización agregando
espacios adicionales o líneas adicionales en tu código HTML.

El navegador eliminará automáticamente cualquier espacio 
o línea adicional cuando la página esté mostrado:

EJEMPLO

<!DOCTYPE html>
<html>
<body>

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser 
ignores it.
</p>

<p>
This paragraph
contains      a lot of spaces
in the source     code,
but the    browser 
ignores it.
</p>

<p>
The number of lines in a paragraph depends on the size of the browser window. 
If you resize the browser window, the number of lines in this paragraph will change.
</p>

</body>
</html>

REGLAS HORIZONTALES HTML

La etiqueta hr define una ruptura tematica en una pagina html y es
la mas frecuente, se muestra como una regla horizontal.

El elemento hr se utiliza para separar contenido en una pagina html.

EJEMPLO

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>

La etiqueta hr es una etiqueta vacia, lo que significa que no tiene etiqueta final.

INTERRUPCION DE LINEA HTML

El elemento br define un salto de linea
Utiliza br si desea un salto de linea sin iniciar un nuevo parrafo.

<p>This is<br>a paragraph<br>with line breaks.</p>

El <br> La etiqueta es una etiqueta vacía, lo que significa que no tiene etiqueta final.

EL PROBLEMA DEL POEMA

Este poema se mostrará en una sola línea:

<p>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</p>

SOLUCION

El elemento "pre" define texto preformateado
El texto dentro del elemento "pre" se muestra en una fuente de anchob fijo
y conserva tanto espacios como saltos de linea.

EJEMPLO

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>

MI EJEMPLO

Este parrafo

esta dentro de un elemento

=== ESTILOS HTML ===

El atributo STYLE se utiliza para agregar estilos a un elemento, como color,
fuente, tamaño y mas.

EL ATRIBUTO STYLE HTML

La configuración del estilo de un elemento HTML se puede realizar con el atributo style.
EL atributo sytile tiene la siguiente sintaxis

<tagname style="property:value;">

El property es una propiedad CSS. El value es un valor CSS.

COLOR DE FONDO

La propiedad CSS background-color define el color de fondo para un elemento de HTML.

EJEMPLO
CAMBIO DE COLOR DE FONDO DE LA PAGINA

<!DOCTYPE html>
<html>

=== FORMATO DE TEXTO HTML ===

HTML contiene varios elementos para definir texto con un 
significado especial.

ELEMENTOS DE FORMATO HTML

<b>- Texto en negrita
<strong>- Texto importante
<i>- Texto en cursiva

<mark>- Texto marcado
<small>- Texto más pequeño
<del>- Texto eliminado
<ins>- Texto insertado
<sub>- Texto de subíndice
<sup>- Texto en superíndice

ELEMENTOS HTML <b> Y <strong>

El elemento <b> define el texto en negrita sin ninguna importancia extra.

=== COMENTARIOS HTML ===

Puede agregar comentarios a su fuente HTML utilizando la siguiente sintaxis:

<!-- Write your comments here -->

Observe que hay un signo de exclamación (!) en la etiqueta de inicio, pero no en la etiqueta final.

AGREGAR COMENTARIOS

Con los comentarios puedes colocar notificaciones y recordatorios en tu código HTML:

EJEMPLO

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

OCULTAR CONTENIDO

Los comentarios se pueden utilizar para ocultar contenido.
Esto puede resultar útil si oculta contenido temporalmente:

EJEMPLO

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph </p> -->

<p>This is a paragraph too.</p>

También puedes ocultar más de una línea. 
Todo entre los <!-- y el --> quedará oculto de la pantalla.

EJEMPLO

<p>This is a paragraph.</p>
<!--
<p>Look at this cool image:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>This is a paragraph too.</p>

Los comentarios también son excelentes para depurar HTML, 
porque puedes comente líneas de código HTML, una a la vez, 
para buscar errores.

OCULTAR CONTENIDO EN LINEA 

Los comentarios se pueden utilizar para ocultar partes en medio del código HTML.

EJEMPLO

<p>This <!-- great text --> is a paragraph.</p>

=== COLORES HTML ===

Los colores HTML se especifican con nombres de colores 
predefinidos o con Valores RGB, HEX, HSL, RGBA o HSLA.

NOMBRES DE COLORES

En HTML, un color se puede especificar utilizando un nombre de color:

Tomate, Naranja, Dodger Azul, Verde marino medio, Gris, Pizarra Azul, Violeta, Gris claro.

Soportes HTML: 140 nombres de colores estandar.

COLOR DE FONDO

Puedes configutar el color de fondo para los elementos HTML.

EJEMPLO

<h1 style="background-color:DodgerBlue;">Hello World</h1>
<p style="background-color:Tomato;">Lorem ipsum...</p>

Puedes configurar el color del texto:

EJEMPLO:

<h1 style="color:Tomato;">Hello World</h1>

VALORES DE COLOR 

En HTML los colores también se pueden especificar 
utilizando valores RGB, valores HEX y HSL valores, 
valores RGBA y valores HSLA.

Los siguientes tres elementos <div> tienen su color 
de fondo establecido con RGB, Valores HEX y HSL:

rgb(255, 99, 71)
#ff6347
hsl(9, 100%, 64%)

<h1 style="background-color:rgb(255, 99, 71);">...</h1>
<h1 style="background-color:#ff6347;">...</h1>
<h1 style="background-color:hsl(9, 100%, 64%);">...</h1>

Los siguientes dos elementos <div> tienen su color de 
fondo establecido con RGBA y valores HSLA, que añaden 
un canal Alfa al color (aquí tenemos el 50% transparencia):

rgba(255, 99, 71, 0,5)
hsla(9, 100%, 64%, 0,5)

<h1 style="background-color:rgba(255, 99, 71, 0.5);">...</h1>
<h1 style="background-color:hsla(9, 100%, 64%, 0.5);">...</h1>

COLORES RGB Y RGBA

Un valor de color RGB representa luz ROJA, VERDE y AZUL fuentes.
Un valor de color RGBA es una extensión de RGB con un Canal alfa (opacidad).

VALORES RGB

En HTML, un color se puede especificar 
como un valor RGB, utilizando esta fórmula:

rgb(rojo, verde, azul)

Cada parámetro (rojo, verde y azul) define la intensidad 
del color con un valor entre 0 y 255.

¡Esto significa que hay 256 x 256 x 256 = 16777216 colores posibles!

Por ejemplo, rgb(255, 0, 0) se muestra en rojo, 
porque el rojo se establece en su valor más alto (255) 
y los otros dos (verde y azul) son establecido en 0.

Otro ejemplo, rgb(0, 255, 0) se muestra en verde, 
porque el verde se establece en su valor más alto (255), 
y los otros dos (rojo y azul) son establecido en 0.

Para mostrar negro, configure todos los parámetros de 
color en 0, de esta manera: rgb(0, 0, 0).

Para mostrar el blanco, configure todos los parámetros 
de color en 255, como esto: rgb(255, 255, 255).
